
# 👨‍👩‍👦‍👦 사부작 👨‍👩‍👦‍👦 
> 사부작 퇴근 후 자기 계발과 취미 생활을 찾아보자!

![](https://github.com/PromiseApp/iOS/assets/99011004/b519d26b-5b9b-48d7-8c40-a6b79f4f7800)

## 📖 Description
사부작은 퇴근 후 자기 계발과 취미 생활을 함께 할 사람을 모집하는 모임 앱입니다.

직장인들이 관심사가 비슷한 사람들을 빠르게 모집하여 취미 생활 및 자기 계발을 함께 할 수 있습니다.

자기만의 취미 생활을 찾지 못한 직장인이 자신에게 맞는 취미를 찾고, 정보를 얻을 수 있습니다.
